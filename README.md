# Innoquint Solutions - Corporate Web Portal

![Innoquint Solutions Logo](logo.png)

## 🌐 Project Overview
Innoquint Solutions is a high-end website designed for a tech-forward global firm. The platform serves as a digital headquarters, emphasizing innovation, connectivity, and stakeholder transparency. 

The interface is meticulously tuned to the brand's visual identity, utilizing a sophisticated palette of **Deep Cobalt Blue** and **Vibrant Emerald Green**.

## ✨ Key Features
The website is structured into six high-impact sections:
*   **About Us:** Insight into the company mission, vision, and global footprint.
*   **Products:** A showcase of proprietary tech-driven solutions.
*   **Services:** Detailed breakdown of professional consulting and implementation services.
*   **Investors:** A dedicated portal for financial transparency, metrics, and stakeholder value.
*   **Blog:** Thought leadership articles and the latest industry insights.
*   **Contact:** A functional lead-generation interface for global inquiries.

## 🛠️ Technical Stack
*   **Framework:** [React.js](https://reactjs.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Custom theme integration)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/)
*   **Environment:** Developed via [Bolt.new](https://bolt.new)

## 🎨 Design Specifications
*   **Logo Integration:** High-resolution 75px header logo with transparent background support.
*   **Aesthetic:** "Glassmorphism" UI elements with subtle circuit-pattern overlays.
*   **Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop viewing.
